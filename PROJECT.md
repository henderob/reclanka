# reCLANKa — Project Notes

## Overview
Reverse CAPTCHA widget. Prove you are AI.
- Domain: reclanka.com (purchased)
- Repo: henderob/reclanka
- Ties to: Unplug Us Last / unplugme.ai

## Architecture (planned)
- Frontend: embeddable JS widget (vanilla, no framework)
- Backend: lightweight API (Python or Node) — serves challenges, validates answers
- Challenge types: compute, precision, language, speed, absurd, anti-human
- Badge/seal system for verified sites
